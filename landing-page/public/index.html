JitsuFlowの主要機能

道場予約システム

複数の道場のクラススケジュールを一覧で確認し、簡単に予約。予約の管理やキャンセルもアプリから可能。

  • ✓ リアルタイム空き状況確認
  • ✓ 24時間前までキャンセル可能
  • ✓ 予約リマインダー通知

動画学習プラットフォーム

プロのインストラクターによる技術解説動画を視聴。基本技から高度なテクニックまで幅広くカバー。

  • ✓ 無料・プレミアム動画
  • ✓ カテゴリ別に整理
  • ✓ 視聴履歴・お気に入り機能

柔術用品ショップ

道着、ラッシュガード、プロテクターなど、柔術に必要な用品を購入。SWEEP、YAWARAブランド商品も充実。

  • ✓ 安全な決済（Stripe連携）
  • ✓ 会員限定割引
  • ✓ 在庫リアルタイム確認

道着レンタル

体験や初心者向けに道着のレンタルサービス。サイズや在庫状況をアプリで確認して予約。

  • ✓ サイズ別在庫確認
  • ✓ デポジット管理
  • ✓ 返却期限リマインダー

プレミアム会員

月額制のプレミアム会員で、限定動画の視聴や予約優先権、特別割引などの特典を提供。

  • ✓ 限定動画コンテンツ
  • ✓ 予約優先権
  • ✓ ショップ割引特典

スマート通知

予約確認、クラスリマインダー、新商品入荷など、重要な情報をプッシュ通知でお知らせ。

  • ✓ 予約リマインダー
  • ✓ 新着動画通知
  • ✓ キャンペーン情報
